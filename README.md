# Credit-Card-Fraud-Detection

A machine learning-based project to detect fraudulent credit card transactions using Python and data science techniques. This system analyzes transaction patterns and identifies suspicious activities to help prevent financial fraud.

📌 Overview

With the increasing use of digital payments, credit card fraud has become a major concern. This project aims to build an intelligent system that can distinguish between legitimate and fraudulent transactions using historical data and predictive modeling.

🚀 Features

📊 Data preprocessing and cleaning

🔍 Exploratory Data Analysis (EDA)

⚖️ Handling imbalanced datasets

🤖 Machine Learning model training

📈 Performance evaluation (Accuracy, Precision, Recall, F1-score)

💡 Fraud detection predictions

🛠️ Tech Stack

Language: Python

⚙️ Installation & Setup

Clone the repository

git clone https://github.com/your-username/credit-card-fraud-detection.git
cd credit-card-fraud-detection

Create virtual environment (optional but recommended)

python -m venv venv
venv\Scripts\activate   # On Windows
source venv/bin/activate  # On Mac/Linux

Install dependencies

pip install -r requirements.txt
▶️ Usage

Run the main script or notebook:

python src/main.py

or open Jupyter Notebook:

jupyter notebook
📊 Model Performance

The model is evaluated using:

Accuracy

Precision

Recall

F1-Score

Special focus is given to Recall, as detecting fraudulent transactions is more critical than minimizing false alarms.
